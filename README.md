## Run Locally

Clone the project

```bash
  git clone https://github.com/manishbajagai2/Meotes.git
```

Go to the project directory

```bash
  cd Meotes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
